## ✦ Project: Fullfocus.

An app built meant to improve user's productivity and focus management to get more things done.

### 🔑 Key Concept

Clean, distraction-free, quick and easy access features to ensure that the user won't  tired from just navigating and using the app. We want to make sure the user can fully distribute their energy to their tasks and projects.

### 🖌️ Visual Style & Experience

We want to keep the visual style as close to Apple's design principal. I found that when navigating through Apple's environment, I feel so fluid and got less distractions so i can fully use my energy to get my things done instead of just trying to get a perfect management everytime I want to start finishing my things.
